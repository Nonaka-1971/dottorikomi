# 仕訳日記帳要素内訳一覧抽出ツール

このプロジェクトは、仕訳日記帳ファイルと要素内訳一覧テンプレートをアップロードすることで、指定された条件に基づいたデータ抽出を行い、結果をダウンロードできるStreamlitアプリケーションです。

## 🚀 機能

- 仕訳日記帳ファイルとテンプレートファイルをアップロード
- モードに基づいたデータ抽出（原価、支払、売上、入金）
- 抽出されたデータをテンプレート形式でダウンロード

## 🛠️ 使用技術

- **Streamlit**: Webアプリケーションの作成
- **Pandas**: Excelデータの処理
- **Openpyxl**: Excelファイルの読み書き
